User Experience Design for Ekart Logistics EPR helping them to manage operations efficiently and reduce the errors

Project Brief

To design User Experience for Logistics & Warehouse Software Application which helps the Manager to Assign Tasks effectively, and the Field Execute to execute tasks diligently with high efficient and less error.

Client

Flipkart

Target Audience

Ekart Admin, Logistics Manager, Hub manage

Date

June- Aug 2017

Design Principles

Information Architecture

IA helped to get all the stakeholders on to same page. This was done by understanding the business/contextual requirements, proposed content for the new system, reading all existing documentation, interview stakeholders and conducting a basic User Research.

How we stood out from the crowd

Current system used by FlipKart has an huge amount of information displayed from various data sources, modules and users. Everyday task and historical data were shown on the same scree. We designed s system that seperates Daily Task and Previous Data. As the end user is not tech savy, he is given a daily task and marked towards completing it. The navigation is design the workflow he executes daily. At end of day he makes progress completing his task

We designed a system that exactly replicates the end users day to day task. The previous days information is kept into historical data. Incomplere tasks move to the next day helping the user to complete his tasks.

Wireframes

All the flows were made in high fidelity wireframes to show the schematic, the skeletal framework of a design were tested with the whole team and then moved on to Interface Design. This helped us for an early feedback for prototype usability tests.

User Interfaces

Minimalism is a perfect marriage of form and function. We designed the screens to have clarity of form ,  clean lines, generous whitespace, and minimal graphical elements bringing simplicity and clarity to even the most confounding subject matter.
